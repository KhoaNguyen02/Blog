Pattern Designs Used

Navigation Tabs

Navigation tabs are positioned below the title of the blog website. The navigation tabs include two main tabs: Home and about. They provide the user with a general overview of the whole website structure. By looking at the navigation tabs, the user is managed to understand the structure of the website quickly and can use it efficiently. In addition, the user can navigate between different pages easily.

Horizontal Dropdown Menu

Horizontal dropdown menu is used to design the sub tabs of each tab in the navigation tabs effectively. For example: when the user click on the Home tab, a dropdown menu will appear which helps them to know which pages belong to the Home page. This type of design not only make full use of the space but also make the whole website look more simple and elegant. Furthermore, this creates a satisfying experience for the user when they need to navigate between too many pages.

Categorization

By having a closer look at the dropdown menu, it is straightforward to realize that there are multiple categorizations. All the contents of the website are classified into these categorizations. As a result, the user can navigate to their desired page without spending too much effort. This indeed enhances the user-friendly of the website.

Article List

At the home page, every articles are displayed in summary. This provide the user with the quickly overview of all posted articles and the abstract of each article. This helps them to briefly know the content of each article so that they can choose their favorite article to read in a short time.
